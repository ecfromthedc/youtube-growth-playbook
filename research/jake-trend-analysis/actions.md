## 2. Ideation lane → `prompts/step2-topic-ideation.md`
Add the **Outlier Transfer** lane:
1. Pull outliers = videos with views >> channel's typical, in adjacent niches.
2. Name the signal (title format / thumbnail / point-count / stakes / raw-vs-polished).
3. Transfer the format to our niche **without changing execution**.
4. Validate against ≥2 prior outliers before greenlighting.

Seed it with the four-format library (speedrun, "I did X N times", "watch me in 10 hours", "[famous person]'s advice on X").

## 3. New skill spec → `rt-yt-skills` (outlier-finder)

- Detect over-performance relative to channel baseline.
- Tag transferable format + packaging signals.
- Output: ranked idea candidates with the "why it worked" rationale.
- Pairs with 1of10 / Spotter Studio already in `docs/03-tool-stack.md`.
