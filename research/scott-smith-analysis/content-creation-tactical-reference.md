## Core Content Formula

### Hook Structure (First 3 seconds)
```
[RESULT] → [METHOD] → [PROOF] → [CTA]
```

**Examples:**
- "I make $100K/month without showing my face"
- "This channel made $20K in 8 months"
- "Want to know the business model everyone's sleeping on?"

### Caption Template

```
👇 [Save/Read instruction]

[Bold claim about the opportunity]

[Why it works / What makes it different]

[Objection handling - "You don't need X"]
⁃ Bullet point 1
⁃ Bullet point 2
⁃ Bullet point 3

[Benefit statement with specific results]

[The gap - what most people don't know]

[Your solution]

[Clear CTA with action word]

#hashtag #hashtag #hashtag
```

## Visual Format Checklist

### Talking Head Setup
- [ ] Simple, clean background
- [ ] Good lighting (natural or ring light)
- [ ] Medium close-up framing
- [ ] Eye contact with camera
- [ ] Confident, energetic delivery

### Text Overlays
- [ ] Bold, high-contrast fonts
- [ ] Yellow/gold for money/results
- [ ] Large enough to read on mobile
- [ ] Emphasize numbers and key words
- [ ] Max 2-3 lines at a time

### Screen Recording
- [ ] Clear, HD quality
- [ ] Highlight important areas
- [ ] Zoom in on key numbers
- [ ] Remove clutter/distractions
- [ ] Add explanatory captions

### B-Roll
- [ ] Match aesthetic to message
- [ ] Use relevant imagery
- [ ] Keep clips 2-3 seconds each
- [ ] Smooth transitions
- [ ] Support the narrative

## Red Flags & Common Mistakes

### Don't Do This
- Generic "make money online" claims
- Fake screenshots or inflated numbers
- Inconsistent posting schedule
- No clear CTA
- Overpromising guaranteed results
- Copying content without adding value
- Ignoring comments/DMs
- Pure clickbait with no substance

### Do This Instead
- Specific niche positioning
- Real, verifiable results
- Daily posting commitment
- Every post has next step
- Transparent about what's required
- Unique voice and perspective
- Engage with your audience
- Deliver actual value in content

## Content Batching Workflow

### Preparation (30 min)
1. Choose 10 topics from content calendar
2. Write 10 hooks
3. Write 10 captions
4. Create shot list

### Recording (2 hours)
1. Set up once
2. Record all 10 reels back-to-back
3. Record different takes of each hook
4. Capture screen recordings if needed

### Editing (3-4 hours)
1. Cut to 30-60 seconds each
2. Add text overlays
3. Add captions/subtitles
4. Color grade for consistency
5. Export optimized for Instagram

### Scheduling (30 min)
1. Upload to scheduler
2. Add final captions
3. Add hashtags
4. Schedule for optimal times
5. Set reminders to engage

**Result:** 10 days of content in one session
